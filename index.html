Sobre Mim

Sou um desenvolvedor web com uma sólida formação em design gráfico. Minha transição para o desenvolvimento web foi impulsionada pelo desejo de unir a estética visual com a funcionalidade técnica, criando experiências digitais que são tanto atraentes quanto eficientes. Com uma compreensão profunda dos princípios de design, sei como construir interfaces intuitivas que não apenas atendem às necessidades dos usuários, mas também refletem a identidade visual de uma marca. Minha experiência em design gráfico me deu uma vantagem única no desenvolvimento front-end, permitindo-me criar layouts responsivos, dinâmicos e visualmente impactantes. Estou sempre em busca de novas oportunidades para expandir minhas habilidades técnicas e explorar as mais recentes tecnologias, com o objetivo de entregar soluções inovadoras e criativas para cada projeto em que trabalho.

Cursos

Full Stack - One Bit Code

O curso Full Stack da One Bit Code oferece uma formação completa em desenvolvimento web, abordando desde o front-end até o back-end. Com foco em tecnologias modernas como React, Node.js e Ruby on Rails, o curso prepara os alunos para criar aplicações web robustas e escaláveis, capacitando-os a atuar em todas as etapas do desenvolvimento de software.

Com aulas que abrangem desde HTML, CSS e JavaScript até frameworks avançados como React e Next.js, o curso também inclui projetos práticos para aplicar o conhecimento adquirido. É uma formação completa que prepara você para criar soluções web inovadoras e atualizadas com as tendências do mercado.

Full Stack - Infinity School

O curso Full Stack da Infinity School oferece uma formação abrangente que combina JavaScript e Python, capacitando você a dominar tanto o front-end quanto o back-end. Além disso, o curso inclui módulos avançados com React, Node.js e outras tecnologias modernas, garantindo uma preparação completa para o desenvolvimento de aplicações web robustas e escaláveis. Ideal para quem busca uma formação sólida e diversificada, com foco em projetos práticos.

Projetos

Calculadora de IMC

Este projeto é uma calculadora de IMC (Índice de Massa Corporal) simples, desenvolvida para ajudar os usuários a calcular e entender seu IMC com base em sua altura e peso. A aplicação permite que o usuário insira seus dados, e com um clique, o resultado é exibido, indicando se o IMC está em uma faixa saudável, abaixo do peso, ou acima do peso.

Cadastro de astronautas

Este projeto é uma landing page simples para inscrições de aspirantes a astronautas. A página inclui um formulário básico para que os usuários possam preencher seus dados e se inscrever rapidamente. O design é limpo e direto, com uma chamada clara para ação, incentivando os visitantes a se tornarem astronautas.

Biblioteca

Este projeto é uma aplicação de biblioteca desenvolvida em Python.
